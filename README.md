# exact_riemann_solvers

Requirements: Octave/gnuplot or Matlab.

* Quick start:

Create local directories
$ mkdir bin && mkdir dat && mkdir err

Invoke Octave
$ octave

Set a problem and run the solver.
$ prob = 'RJ2a';
$ exact_mhd_run;

Print the solution.
$ print_states_mhd;

Plot the solution.
$ plot_mhd;

Plot the perpendicular magnetic field.
$ plot_bperp;
